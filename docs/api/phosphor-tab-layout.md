# Phosphor Tab Layout

### Attributes / Events / Properties

### Attributes

- `data-tabs-movable`

### Events
- `tab-change`

## Usage

```html
  <phosphor-tab-layout>
    <phosphor-widget data-title="Tab 1">
      <!-- Your Content Goes here -->
    </phosphor-widget>
    <phosphor-widget data-title="Tab 2" data-closable="true">
      <!-- Your Content Goes here -->
    </phosphor-widget>
  </phosphor-tab-layout>
```
